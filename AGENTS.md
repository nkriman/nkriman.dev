# AGENTS.md

Personal blog. Astro 5, content collections, no client JS.

- Dev: `npm run dev` · Build: `npm run build` (output in `dist/`)
- Posts: markdown in `src/content/blog/` with title/description/pubDate frontmatter; `draft: true` hides a post.
- Styling lives in `src/styles/global.css` (typography-first, light/dark via prefers-color-scheme).
- Deploys via Vercel Git integration on push to main.
